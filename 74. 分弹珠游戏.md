# 分弹珠游戏（1005）

## 题目描述

Alice最近迷上分弹珠游戏，每次从箱子里面拿出随机数量的弹珠，第一次将弹珠分为尽可能均匀的两份（偶数均分，奇数则其中一份比另外一份多一颗），之后将这两份中每份的数量继续拆分成两份，依然尽可能的均匀，直到某一份的弹珠不超过1颗为止。如果第一次取出弹珠就已经≤1颗，则不再拆分。你能在Alice取出弹珠后就知道它最终会分多少次、最终会分成多少份吗？

## 输入描述

一个整数N，表示Alice拿到的弹珠数。
范围：0 < N < 131072

## 输出描述

一行数据，表示分拆后的次数数；一个整数，表示分完后成的份数。

## 示例

### 示例1
**输入**
```
1
```

**输出**
```
0 1
```

### 示例2
**输入**
```
11
```

**输出**
```
6 7
```

**说明**：11颗第一次拆分为5和6。第二次将5拆分为2和3。第三次将6拆分为3和3。每个3会再拆分1次。共让拆分6次，分为7份。
