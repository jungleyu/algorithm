# 字符串摘要

## 题目描述

给定一个字符串的摘要算法，请输出给定字符串的摘要值。

**处理规则：**
1. **去除非字母符号**：去除字符串中非字母的符号
2. **连续字符处理**：如果出现连续字符（不区分大小写），则输出：该字符（小写）+ 连续出现的次数
3. **非连续字符处理**：如果是非连续的字符（不区分大小写），则输出：该字符（小写）+ 该字母之后字符串中出现的该字符的次数
4. **排序规则**：对按照以上方式表示后的字符串进行排序
   - 字母和紧随的数字作为一组进行排序
   - 数字大的在前
   - 数字相同的，则按字母进行排序，字母小的在前

## 输入描述

一行字符串，长度为 `[1, 200]`

## 输出描述

摘要字符串

## 示例

### 示例1

**输入**
```
aabbcc
```

**输出**
```
a2b2c2
```

**说明**
- 无

---

### 示例2

**输入**
```
bAaAcBb
```

**输出**
```
a3b2b2c0
```

**说明**
- `bAaAcBb`：
  - 第一个 `b` 非连续字母，该字母之后字符串中还出现了 2 次（最后的两个 `Bb`），所以输出 `b2`
  - `a` 连续出现 3 次，输出 `a3`
  - `c` 非连续，该字母之后字符串再没有出现过 `c`，输出 `c0`
  - `Bb` 连续 2 次，输出 `b2`
- 对 `b2a3c0b2` 进行排序，最终输出 `a3b2b2c0`
