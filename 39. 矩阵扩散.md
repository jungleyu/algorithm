# 矩阵扩散

## 题目描述

存在一个 m×n 的二维数组，其成员取值范围为 0 或 1。

其中值为 1 的成员具备扩散性，每经过 1S，将上下左右值为 0 的成员同化为 1。

二维数组的成员初始值都为 0，将第 [i,j] 和 [k,l] 两个位置上元素修改成 1 后，求矩阵的所有元素变为 1 需要多长时间。

## 输入描述

输入数据格式为：`m,n,i,j,k,l`
- 前 2 个数字表示这是一个 m×n 的矩阵，m 和 n 不会超过 1024 大小；
- 中间两个数字表示一个初始扩散点位置为 i,j；
- 最后 2 个数字表示另一个初始扩散点位置为 k,l。

## 输出描述

输出矩阵的所有元素变为 1 所需要的秒数。

## 示例

### 示例 1

**输入**
```
4,4,0,0,3,3
```

**输出**
```
3
```

**说明**

- 输入数据中的前 2 个数字表示这是一个 4×4 的矩阵；
- 中间两个数字表示一个初始扩散点位置为 (0,0)；
- 最后两个数字表示另一个初始扩散点位置为 (3,3)；
- 初始点在对角线上，达到中间的位置分别需要 3 次迭代，即 3 秒。
