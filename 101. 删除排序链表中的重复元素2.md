# 100. 删除排序链表中的重复元素 II

给定一个已排序的链表的头 `head` ，**删除原始链表中所有重复数字的节点，只留下不同的数字**。返回 **已排序的链表**。

## 示例 1：

链表示意图：

```
1 -> 2 -> 3 -> 3 -> 4 -> 4 -> 5 -> null
```

**输入：**

```
head = [1,2,3,3,4,4,5]
```

**输出：**

```
[1,2,5]
```

**解释：** 链表中 3 和 4 出现了多次，需要全部删除，只保留出现一次的数字。

## 示例 2：

链表示意图：

```
1 -> 1 -> 1 -> 2 -> 3 -> null
```

**输入：**

```
head = [1,1,1,2,3]
```

**输出：**

```
[2,3]
```

**解释：** 链表中 1 出现了多次，需要全部删除，只保留出现一次的数字。

## 提示：

- 链表中节点数目在范围 `[0, 300]` 内
- `-100 <= Node.val <= 100`
- 题目数据保证链表已经按升序排列
