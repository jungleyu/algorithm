# 智能驾驶

## 题目描述

有一辆汽车需要从 `M × N` 的地图左上角（起点）开往地图的右下角（终点），去往每一个地区都需要消耗一定的油量，加油站可进行加油。

请你计算汽车确保从起点到达终点时所需的最少初始油量。

**移动规则**：
- 智能汽车可以上下左右四个方向移动
- 地图上的数字取值是 `0`、`-1` 或正整数

**地图含义**：

| 值 | 含义 |
|---|---|
| `-1` | 加油站，可以加满油，汽车的油箱容量最大为100 |
| `0` | 障碍物，汽车不能通过 |
| 正整数 | 汽车走过这个地区的耗油量 |

**特殊情况**：
- 如果汽车无论如何都无法到达终点，则返回 `-1`

## 输入描述

第一行为两个数字 `M` 和 `N`，表示地图的大小为 `M × N`

- `0 < M, N ≤ 200`
- 后面是一个 `M × N` 的矩阵，其中的值是 `0` 或 `-1` 或正整数
- 加油站（`-1`）的总数不超过 200 个

## 输出描述

- 如果汽车无论如何都无法到达终点，则返回 `-1`
- 如果汽车可以到达终点，则返回最少的初始油量

## 示例

### 示例1

**输入**
```
2,2
10,20
30,40
```

**输出**
```
70
```

**说明**
行走的路线为：右→下

---

### 示例2

**输入**
```
4,4
10,30,30,20
30,30,-1,10
0,20,20,40
10,-1,30,40
```

**输出**
```
70
```

**说明**
行走的路线为：右→右→下→下→下→右

---

### 示例3

**输入**
```
4,5
10,0,30,-1,10
30,0,20,0,20
10,0,10,0,30
10,-1,30,0,10
```

**输出**
```
60
```

**说明**
行走的路线为：下→下→下→右→右→上→上→上→右→右→下→下→下

---

### 示例4

**输入**
```
4,4
10,30,30,20
30,30,20,10
10,20,10,40
10,20,30,40
```

**输出**
```
-1
```

**说明**
无论如何都无法到达终点
