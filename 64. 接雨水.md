# 接雨水

## 题目描述
给定 `n` 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

## 输入描述
输入为一个整数数组 `height`，表示柱子的高度。

**格式**：
```
n
height[0] height[1] ... height[n-1]
```

**说明**：
- 第 1 行：一个整数 `n`，表示柱子的数量
- 第 2 行：`n` 个非负整数，表示每个柱子的高度

## 输出描述
输出一个整数，表示能接的雨水总量。

**格式**：
```
result
```

**说明**：
- 输出为一个整数，表示下雨后能接的雨水总量

## 补充说明
- 1 <= n <= 2 * 10^4
- 0 <= height[i] <= 10^5

## 示例

### 示例1
**输入**：
```
12
0 1 0 2 1 0 1 3 2 1 2 1
```

**输出**：
```
6
```

**说明**：
输入数组表示的高度图如下：
```
      3
    2   2
  1 2 1 2
0 1 0 2 1 0 1 3 2 1 2 1
```
蓝色部分表示雨水，总共可以接 6 个单位的雨水。

### 示例2
**输入**：
```
6
4 2 0 3 2 5
```

**输出**：
```
9
```

**说明**：
输入数组表示的高度图如下：
```
      5
    3   5
  2 3 2
4 2 0 3 2 5
```
蓝色部分表示雨水，总共可以接 9 个单位的雨水。

### 示例3
**输入**：
```
3
1 2 1
```

**输出**：
```
0
```

**说明**：
三个柱子呈 V 字形，无法接住任何雨水。
