# 周末爬山

## 题目描述

周末小明准备去爬山锻炼，地图中：
- `0` 代表平地
- 山的高度使用 `1~9` 来表示

小明的移动规则：
- 每次爬山或下山高度差只能相差 `k` 及 `k` 以内
- 每次只能上下左右一个方向移动一格
- 小明从左上角 `(0,0)` 位置出发

## 输入描述

第一行输入 `m n k`（空格分隔）：
- `m` × `n` 的二维山地图
- `k` 为每次爬山或下山高度差的最大值

接下来输入山地图，共 `m` 行 `n` 列，均以空格分隔。

**取值范围**：
- `0 < m ≤ 500`
- `0 < n ≤ 500`
- `0 < k < 5`

**备注**：所有用例输入均为正确格式，考生不需要考虑不合法的输入格式。

## 输出描述

输出小明能爬到的**最高峰高度**和到该最高峰的**最短步数**，以空格分隔。

**规则**：
- 同高度的山峰输出较短步数
- 如果没有可以爬的山峰（所有可达位置都是平地），则高度和步数都返回 `0`

## 示例

### 示例1

**输入**
```
5 4 1
0 1 2 0
1 0 0 0
1 0 1 2
1 3 1 0
0 0 0 9
```

**输出**
```
2 2
```

**说明**
- 能爬到的最高峰在 `(0,2)` 位置，高度为 `2`
- 最短路径：`(0,0) → (0,1) → (0,2)`
- 最短步数：`2`

### 示例2

**输入**
```
5 4 3
0 0 0 0
0 0 0 0
0 9 0 0
0 0 0 0
0 0 0 9
```

**输出**
```
0 0
```

**说明**
- 每次爬山高度差最大为 `3`
- 从平地 `0` 无法爬到高度为 `9` 的山峰（高度差为 `9 > 3`）
- 没有可以爬的山峰，输出 `0 0`
