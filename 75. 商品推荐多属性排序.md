# 商品推荐多属性排序（100分）

## 题目描述

双十一来临，小王想在某个购物平台买坚果。在输入坚果时，页面默认按照综合推荐排序列表。小王很想知道这个综合排序是如何完成的，于是便开始思考算法。经过一番操作后，完成了他的多个关键字排序算法。希望你能给小王提供一份正确的排序结果，验证其实现算法结果的准确性。

## 输入描述

第一行两个整数k,n（m≤10，n≤100000），分别表示有n个需要排序的商品。每个商品有m个属性。

第二行m个整数（均为1或-1），每个整数表示第i个属性按降序（1）或是升序排列（-1），依次按照属性1~m排序。

接下来n行，每行m个整数，表示每个商品的m个属性。

## 输出描述

输出排序后的结果。

## 补充说明

商品评价属性相同时，按照下个属性维度排序。

## 示例

### 输入
```
4 3
1 -1 1
2 2 2
2 3 3
4 4 4
4 4 5
```

### 输出
```
4 4 5
4 4 4
2 2 2
2 3 3
```
