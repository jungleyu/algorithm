# 流量波峰

## 题目描述
一线运维人员在对通话流量进行监控，每一段时间内都会出现流量的高峰，流量有高有低形成一个个波峰波谷。运维人员想找到流量变化最快的波峰，请帮助他找到满足以下条件的三元组 `(i, j, k)`：

其中 `i < j < k`，`nums[i] < nums[j]` 且 `nums[j] > nums[k]`（即 `j` 是峰顶），并找到所有满足条件的三元组中 `(k - i)` 的最小值。

## 输入描述
第一行为 `n` 个整数，表示数组中的 `n` 个元素，`0 <= n <= 100000`

## 输出描述
返回所有满足条件的三元组中 `(k - i)` 的最小值。若不存在，返回 `-1`。

## 示例

### 示例1
**输入**：`3 5 4 7 2 1`

**输出**：`2`

**说明**：满足条件的三元组为 `[0, 1, 2]`，距离为 `2`

### 示例2
**输入**：`4 3 2 1`

**输出**：`-1`

**说明**：无法找到满足条件的三元组，返回 `-1`
