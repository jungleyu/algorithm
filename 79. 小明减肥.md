# 小明减肥

## 题目描述

小明最近想通过运动减肥。他列出了 n 种不同的运动，1 <= n <= 10。每种运动消耗的卡路里分别为 calories[i]，calories[i] >= 0。他希望选择恰好 K 种运动进行锻炼，使得这些运动消耗的总卡路里恰好等于目标 T，T >= 0。注意，每种运动最多只能选一次，且 K >= 0。请你帮助小明计算，共有多少种不同的运动组合方式可以达到目标。两种组合不同当且仅当至少包含一个不同的运动。

## 输入描述

第一行三个整数 n、T、K，表示运动的总数量、目标卡路里和需要选择的运动数量。

第二行 n 个整数，表示每种运动的消耗。

## 输出描述

一个整数，表示不同的组合数目。

## 示例 1

### 输入
```
5 6 3
1 2 3 4 5
```

### 输出
```
1
```

### 说明
唯一满足条件的组合是『1, 2, 3』（总和为 6，恰好选 3 个运动）。

## 示例 2

### 输入
```
7 7 3
0 1 2 3 4 5 6
```

### 输出
```
4
```

### 说明
满足条件的组合是『0,1,6』『0,2,5』『0,3,4』『1,2,4』（总和为 7，恰好选 3 个运动）。
