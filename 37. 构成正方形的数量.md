# 构成正方形的数量

## 题目描述

输入N个互不相同的二维整数坐标，求这N个坐标可以构成的正方形数量。

> 补充说明：内积为零的两个向量垂直

## 输入描述

第一行输入为 N，N 代表坐标数量，N 为正整数。N ≤ 100

之后的 K 行输入为坐标 x y，以空格分隔，x，y 为整数，-10 ≤ x, y ≤ 10

## 输出描述

输出可以构成的正方形数量。

## 示例

### 示例1

**输入**

```
3
1 3
2 4
3 1
```

**输出**

```
0
```

**说明**

3个点不足以构成正方形。

### 示例2

**输入**

```
4
0 0
1 2
3 1
2 -1
```

**输出**

```
1
```

**说明**

这4个点可以构成一个正方形。
