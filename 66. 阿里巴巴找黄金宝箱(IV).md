# 阿里巴巴找黄金宝箱(IV)

## 题目描述
一贫如洗的樵夫阿里巴巴在去砍柴的路上，无意中发现了强盗集团的藏宝地。藏宝地有编号从 0-N 的箱子，每个箱子上面有一个数字，箱子排列成一个环，编号最大的箱子的下一个是编号为 0 的箱子。

请输出每个箱子贴的数字之后的第一个比它大的数，如果不存在则输出 -1。

## 输入描述
输入一个数字字符串，数字之间使用逗号分隔，例如：`1,2,3,1`

**范围**：
- 字符串中数字个数：1 ≤ n ≤ 10000
- 每个数字值：-100000 ≤ value ≤ 100000

## 输出描述
下一个更大数的列表，以逗号分隔，例如：`2,3,6,-1,6`

## 示例

### 示例1
**输入**：`2,5,2`

**输出**：`5,-1,5`

**说明**：
- 第一个 2 的下一个更大的数是 5
- 数字 5 找不到下一个更大的数，输出 -1
- 第二个 2 的下一个最大的数需要循环搜索，结果也是 5

### 示例2
**输入**：`3,4,5,6,3`

**输出**：`4,5,6,-1,4`
