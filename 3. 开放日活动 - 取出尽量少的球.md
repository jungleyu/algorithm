# 开放日活动 - 取出尽量少的球

## 题目描述

某部门开展 Family Day 开放日活动，其中有个从桶里取球的游戏，游戏规则如下：

- 有 `N` 个容量一样的小桶等距排开
- 每个小桶都默认装了数量不等的小球，记录在数组 `bucketBallNums` 中
- 游戏开始时，要求所有桶的小球总数不能超过 `SUM`
- 如果小球总数超过 `SUM`，则需对所有小桶统一设置一个容量最大值 `maxCapacity`，并将超过容量最大值的小球拿出来，直至小桶里的小球数量小于 `maxCapacity`

请根据输入的数据，计算从每个小桶里拿出的小球数量。

### 限制规则

**规则一**：如果所有小桶的小球总和小于 `SUM`，则无需设置容量值 `maxCapacity`，无需拿球，返回 `[]`

**规则二**：如果所有小桶的小球总和大于 `SUM`，则需设置容量最大值 `maxCapacity`，并从小桶中拿球，返回从每个小桶拿出的小球数量组成的数组

## 输入描述

- 第一行：输入 2 个正整数，数字之间使用空格隔开
  - 第一个数字表示 `SUM`
  - 第二个数字表示 `bucketBallNums` 数组长度
- 第二行：输入 `N` 个正整数，数字之间使用空格隔开，表示 `bucketBallNums` 的每一项

## 输出描述

找到一个 `maxCapacity`，保证取出尽量少的球，并返回从每个小桶拿出的小球数量组成的数组。

## 备注

- `1 ≤ bucketBallNums[i] ≤ 10^9`
- `1 ≤ bucketBallNums.length = N ≤ 10^5`
- `1 ≤ maxCapacity ≤ 10^9`
- `1 ≤ SUM ≤ 10^9`

## 示例

### 示例 1

**输入**：
```
14 7
2 3 2 5 5 1 4
```

**输出**：
```
[0,1,0,3,3,0,2]
```

**说明**：小球总数为 22，`SUM=14`，超出范围需取球：
- `maxCapacity=1`：剩余总和为 7（远小于 14）
- `maxCapacity=2`：剩余总和为 13
- `maxCapacity=3`：剩余总和为 16（大于 14）

因此 `maxCapacity=2`，每个小桶小球数量大于 2 的都需要拿出。

### 示例 2

**输入**：
```
3 3
1 2 3
```

**输出**：
```
[0,1,2]
```

**说明**：小球总数为 6，`SUM=3`，超出范围需取球。`maxCapacity=1`，剩余总和为 3。

### 示例 3

**输入**：
```
6 2
3 2
```

**输出**：
```
[]
```

**说明**：小球总数为 5，`SUM=6`，在范围内，无需取球。

## 解题思路

1. **计算总和**：首先计算所有桶的小球总数
2. **判断是否需要取球**：如果总和 ≤ SUM，直接返回空数组
3. **二分查找 maxCapacity**：在 `[0, max(bucketBallNums)]` 范围内二分查找最大的 `maxCapacity`，使得限制后的总和 ≤ SUM
4. **计算结果**：根据找到的 `maxCapacity`，计算每个桶需要拿出的球数
