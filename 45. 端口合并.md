# 端口合并

## 题目描述

有 M 个端口组（1 ≤ M ≤ 10），每个端口组是长度为 N 的整数数组（1 ≤ N ≤ 100）。

**合并规则**：如果端口组间存在 2 个及以上不同端口相同，则认为这 2 个端口组互相关联，可以合并。

## 输入描述

- 第一行输入端口组个数 M
- 接下来输入 M 行，每行用逗号分隔，代表端口组
- **备注**：端口组内数字可以重复

## 输出描述

输出合并后的端口组，用二维数组表示：
- 组内相同端口仅保留一个，从小到大排序
- 组外顺序保持输入顺序

**异常情况**：若 M、N 不在限定范围内，统一输出一组空数组 `[[]]`

## 示例

### 示例 1

**输入**
```
4
4
2,3,2
1,2
5
```

**输出**
```
[[4],[2,3],[1,2],[5]]
```

**说明**：仅有一个端口 2 相同，不可以合并。

### 示例 2

**输入**
```
3
2,3,1
4,3,2
5
```

**输出**
```
[[1,2,3,4],[5]]
```

**说明**：前两个端口组有 2、3 两个共同端口，可以合并。

### 示例 3

**输入**
```
6
10
4,2,1
9
3,6,9,2
6,3,4
8
```

**输出**
```
[[10],[1,2,3,4,6,9],[9],[8]]
```

**说明**：第2、4、5组可以合并（通过共同端口关联）。

### 示例 4

**输入**
```
11
```

**输出**
```
[[]]
```

**说明**：M=11 超出范围（1 ≤ M ≤ 10）。
