# 停车场收入统计

## 题目描述

给定一个停车场某一天的车辆出入记录，请计算该停车场的当日收入，收费规则如下：

- 停车半小时收费1元，不足半小时按半小时计算。
- 一辆车每天收费封顶15元。
- 停车时间小于半小时则不收费。
- 11:30 - 13:30 不计入停车时间。
- 包月的车辆不统计当日收入。

## 输入描述

输入分两段。

第一段是购买了包月服务的车辆信息：第一行是包月车辆的数量；从第二行开始，每行是一个包月的车牌号。

第二段是当日车辆出入记录：格式为：`时:分 车牌号 enter/leave`

## 输出描述

停车场当日收入，整数。

## 补充说明

假设所有车辆在当天进当天离。每天的车辆出入记录 < 10000 条，包月车辆个数小于 10000。

## 示例 1

### 输入
```
1
A12345
01:28 A12345 enter
01:59 A12345 leave
```

### 输出
```
0
```

## 示例 2

### 输入
```
0
08:00 A12346 enter
11:20 A12345 enter
11:30 A12346 leave
12:00 A12345 leave
```

### 输出
```
7
```

---

**时间限制**：1秒
**空间限制**：262MB
