# 文本统计分析

## 题目描述

有一个文件，包含以一定规则写作的文本，请统计文件中包含的**语句数量**。

### 规则说明

**1. 语句分隔**
- 文本以 `;` 分隔
- 最后一条语句可以没有 `;`
- **空文本不能算语句**（无字符、空白字符、制表符都算作空）
- 示例：`"COMMAND A; ;"` 只能算一条语句

**2. 跨行文本**
- 文本可以跨行，下面是**一条**文本：
```
COMMAND A
AND
COMMAND B;
```

**3. 字符串支持**
- 字符串为成对的单引号 `'` 或双引号 `"`
- 支持转义字符 `\` 处理单双引号和转义字符本身
- 示例：
```
COMMAND A "Say \"hello\"";
```

**4. 注释支持**
- 注释以 `--` 开头，到换行结束
- 注释可以出现在字符串之外的任意位置
- **字符串内的 `--` 不是注释**
- 示例：
```
COMMAND A; --this is comment
COMMAND --comment
A AND COMMAND B;
```

## 输入描述

文本文件内容

## 输出描述

输出文件中包含的语句数量

## 示例

**输入**
```
COMMAND TABLE IF EXISTS "UNITED STATE";
COMMAND A GREAT (
ID ADSAB,
download_length INTE-GER, -- test
file_name TEXT,
guid TEXT,
mime_type TEXT,
notifica-tionid INTEGER,
original_file_name TEXT,
pause_reason_type INTEGER,
resumable_flag INTEGER,
start_time INTEGER,
state INTEGER,
folder TEXT,
path TEXT,
total_length INTE-GER,
url TEXT
);
```

**输出**
```
2
```

**说明**
- 第一条语句：`COMMAND TABLE IF EXISTS "UNITED STATE";`
- 第二条语句：`COMMAND A GREAT (...)`（跨行语句，以 `);` 结尾）
- 注释 `-- test` 被忽略
