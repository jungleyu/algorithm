# 比赛

## 题目描述

有 N 个选手参加比赛，选手编号为 1~N（3 ≤ N ≤ 100），有 M（3 ≤ M ≤ 10）个评委对选手进行打分。

**打分规则**：
- 每个评委对选手打分，最高分 10 分，最低分 1 分
- 计算得分最多的 3 位选手的编号
- **得分相同时**：得分高分值最多的选手排名靠前（10分数量相同则比较9分，以此类推）
- 用例中不会出现多个选手得分完全相同的情况

## 输入描述

第一行为半角逗号分割的两个正整数：
- 第一个数字表示 M（评委数量），3 ≤ M ≤ 10
- 第二个数字表示 N（选手数量），3 ≤ N ≤ 100

第 2 到 M+1 行是半角逗号分割的整数序列，表示评委为每个选手的打分：
- 0 号下标数字表示 1 号选手分数
- 1 号下标数字表示 2 号选手分数，依次类推

## 输出描述

输出选手前 3 名的编号（逗号分隔）。

**异常情况**：若输入为异常，输出 `-1`，如：
- M 不在 [3, 10] 范围内
- N 不在 [3, 100] 范围内
- 打分不在 [1, 10] 范围内

## 示例

### 示例 1

**输入**
```
4,5
10,6,9,7,6
9,10,6,7,5
8,10,6,5,10
9,10,8,4,9
```

**输出**
```
2,1,5
```

**说明**
- 2 号选手得分 36 分（10 分有 3 个）排第 1
- 1 号选手得分 36 分（10 分有 1 个）排第 2
- 5 号选手得分 30 分排第 3

### 示例 2

**输入**
```
2,5
7,3,5,4,2
8,5,4,4,3
```

**输出**
```
-1
```

**说明**：只有 2 个评委，要求最少为 3 个评委。

### 示例 3

**输入**
```
4,2
8,5
5,6
10,4
8,9
```

**输出**
```
-1
```

**说明**：只有 2 名选手参加，要求最少为 3 名。

### 示例 4

**输入**
```
4,5
11,6,9,7,8
9,10,6,7,8
8,10,6,9,7
9,10,8,6,7
```

**输出**
```
-1
```

**说明**：第一个评委给第一个选手打分 11，无效分数。
