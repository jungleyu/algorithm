# 考勤信息

## 题目描述

公司用一个字符串来表示员工的出勤信息：

- `absent`：缺勤
- `late`：迟到
- `leaveearly`：早退
- `present`：正常上班

现需根据员工出勤信息，判断本次是否能获得出勤奖，能获得出勤奖的条件如下：

1. 缺勤不超过一次；
2. 没有连续的迟到/早退；
3. 任意连续7次考勤，缺勤/迟到/早退不超过3次。

## 输入描述

第一行输入一个整数 N，表示考勤记录的条数。

之后 N 行，每行输入一个字符串，表示该员工的考勤数据，各条考勤记录用空格分隔。

> 约束条件：
> - 记录条数 >= 1
> - 输入字符串长度 < 10000
> - 不存在非法输入

## 输出描述

输出 N 个结果，每个结果为 `"true"` 或 `"false"`，表示对应员工是否能获得考勤奖。

## 示例

### 示例1

**输入**

```
2
present
present present
```

**输出**

```
true true
```

**说明**

两个员工都满足出勤奖条件。

### 示例2

**输入**

```
2
present
present absent present present leaveearly present absent
```

**输出**

```
true false
```

**说明**

第一个员工只有一次正常出勤，满足条件；第二个员工缺勤两次（超过一次），不满足条件。
