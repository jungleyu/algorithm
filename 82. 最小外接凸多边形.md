# 200分 最小外接凸多边形
思路待更新

## 题目描述
最小外接凸多边形(又叫凸包)是包含所有输入的多边形的最小凸多边形。简单来讲,可以把最小外接凸多边形看作是用橡皮筋套住所有输入的多边形上而获得的新多边形。在网络规划、网络优化、地理化呈现等领域或有许多应用。

## 输入
多个point(xy)点集合,每个集合内的顶点point顺序相连构成一个多边形polygon
对每个输入:第一行内有整数n,表示接下来有多少个多边形
下面每行为8个整数,空格分隔,表示多边形的4个顶点坐标xy
0<输入的多边形polygon个数<20
每个顶点的坐标(xy)为整数,范围0<=xy<=100
输入的多边形均为凸四边形

## 输出描述
输出:一个point点集合,表示输入的最小外接凸多边形。最左侧(x最小,x相等时y最小)为顶点,逆时针相连。输出中每个数字空格分隔。形如x1y1x2y2x3y3x4 y4...n yn

## 补充说明
提示:凸多边形有一条特性,任意相邻的两个顶点的连线,所有其余点总在其一侧。
可通过计算从A、B到某一C点的向量叉积cross判断相对位置
假设O即为坐标原点(x0,y0),A点的坐标为(x1.y1),B点的坐标为(x2.y2),
那么,向量叉积cross=(x1-x0)(y2-y0)-(y1-y0)(x2-x0)
cross<0说明O在OAB组成的三角形的顺时针方向
cross=0表示O、A、B在同一条直线上
cross>0表示O在OAB组成的逆时针方向

## 示例1
### 输入
```
2
0 2 0 0 2 0 2 2
1 4 1 1 3 1 3 4
```
### 输出
```
0 0 2 0 3 1 3 4 1 4 0 2
```
### 说明
输入为正方形ABCD和长方形EFGH,输出为多边形BCGHEA
