# 恢复数字序列

## 题目描述

对于一个连续正整数组成的序列，可以将其拼接成一个字符串，再将字符串里的部分字符打乱顺序。如序列 `8 9 10 11 12`，拼接成的字符串为 `89101112`，打乱一部分字符后得到 `90811211`，原来的正整数 `10` 就被拆成了 `0` 和 `1`。

现给定一个按如上规则得到的打乱字符的字符串，请将其还原成连续正整数序列，并输出序列中最小的数字。

## 输入描述

输入一行，为打乱字符的字符串和正整数序列的长度，两者间用空格分隔。

**输入约束**：
- 字符串长度不超过 200
- 正整数序列长度不超过 1000
- 保证输入可以还原成唯一序列

## 输出描述

输出一个数字，为序列中最小的数字。

## 示例

### 示例1

**输入**：
```
19801211 5
```

**输出**：
```
8
```

**说明**：
还原后的序列为 `8, 9, 10, 11, 12`，最小数字为 `8`。

### 示例2

**输入**：
```
432111111111 4
```

**输出**：
```
111
```

**说明**：
还原后的序列为 `111, 112, 113, 114`，最小数字为 `111`。
